Tools mentioned in the proceedings of the annual ADHO conferences (2015–2019)

(DH2015, Sydney; DH2016, Kraków; DH2017, Montréal; DH2018, Ciudad de México; DH2019, Utrecht)

This is a little exercise to understand which of the tools in the Digital Humanities are actually being used to do research. We chose the DH conferences because they are the biggest and broadest events in the field, and also because the data was freely available (CC BY 4.0). For data extraction we wrote a little Java tool called ToolXtractor. It uses a list of tools and tries to match them with the full texts of the proceedings. Since this is simple string matching, we had to throw out some false positives manually (not too many). The list we used was gathered via TAPoR which features around 1.500 tools in its database. We kept filenames as they were. With the exception of DH2017, all files are in TEI format.

Corresponding blog post: Which DH Tools Are Actually Used in Research? (some more info and visualisations)
